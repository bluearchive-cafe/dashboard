# resource-panel

用于管理游戏内汉化资源的控制面板页面。

## 这是什么

这是一个用于查看和管理游戏内汉化资源的网页控制面板。你可以通过它查看当前资源状态，切换部分资源开关，并将设置保存到服务器。

页面支持以下功能：

- 查看游戏文本、主线中配、图像视频三类资源的状态
- 开启或关闭对应资源
- 保存当前设置
- 查看操作说明
- 复制当前控制面板链接
- 查看诊断信息，便于排查问题

## 如何打开

推荐使用主页面：

- `index.html`

如果你拿到的是一个完整链接，直接在浏览器中打开即可，比如：

```text
https://dash.bluearchive.cafe/?uid=
```

`?uid=` 后方填入你在游戏内获得的 UID 即可使用。

如果你是本地打开页面，请尽量使用带有 `uid` 参数的地址，例如：

```text
index.html?uid=XXXXXXXX
```

或：

```text
http://127.0.0.1:8080/index.html?uid=XXXXXXXX
```

## 使用前请注意

- 页面依赖网络连接才能正常使用
- 页面会访问远程接口读取和保存配置
- 页面样式依赖在线资源，网络受限时可能显示异常
- 如果没有提供有效的 `uid`，页面无法读取配置，也无法保存设置

## 页面怎么用

### 1. 打开页面后先看状态

页面打开后会自动读取当前配置，并显示三类资源的状态：

- 游戏文本
- 主线中配
- 图像视频

状态说明：

- `加载中`：正在读取数据
- `可用`：当前资源状态正常
- `待更新`：资源可能需要更新后才能正常使用
- `获取失败`：页面未能成功读取状态

建议先确认状态，再决定是否开启对应功能。

### 2. 按需切换开关

页面中有三个对应开关：

- 游戏文本
- 主线中配
- 图像视频

你可以根据需要打开或关闭它们。

### 3. 点击“保存设置”

调整完成后，点击 `保存设置`，页面会将当前开关状态提交到服务器。

保存成功时会提示：

```text
设置已保存，重启游戏后生效
```

如果保存失败，页面会弹出相应提示。

### 4. 点击“查看说明”

如果你不确定每个选项的作用，可以点击 `查看说明`。页面会弹出简要操作提示。

### 5. 点击“复制链接”

点击后，页面会尝试复制当前控制面板链接，方便你发给别人或自己留存。

链接一般类似：

```text
https://dash.bluearchive.cafe/?uid=XXXXXXXX
```

只有当前页面带有有效 `uid` 参数时，才能正确生成可分享链接。

### 6. 点击“诊断信息”

如果页面异常、状态获取失败或保存失败，可以点击 `诊断信息` 查看当前环境信息，用于排查问题。

诊断信息通常包括：

- 浏览器版本信息
- 当前设备和语言信息
- 页面地址
- 资源版本信息
- 屏幕与视口尺寸

## 常见问题

### 页面提示“链接无效”

这通常表示当前页面地址中缺少 `uid` 参数，或 `uid` 参数无效。

请检查链接中是否包含类似：

```text
?uid=XXXXXXXX
```

### 状态一直显示获取失败

可能是以下原因之一：

- 浏览器无法访问远程 API
- 当前网络环境限制了接口访问
- 接口服务异常

### 复制链接失败

常见原因：

- 当前页面没有有效 `uid`
- 浏览器限制了剪贴板写入权限
- 当前运行环境对 `file://` 页面支持较差

### 保存设置失败

可能是以下原因之一：

- 当前网络异常
- 接口服务暂时不可用
- 当前链接中的 `uid` 无效

## 给普通用户的建议

- 尽量使用别人提供的完整控制面板链接直接打开
- 修改设置前，先确认三类资源状态是否正常
- 保存成功后，按页面提示重启游戏再确认效果
- 如果遇到异常，优先查看“诊断信息”并保留当前链接
